We Also Got Boring Code Snippets

There's no syntax highlighting, but there is a vanilla block when you want to display some code:

def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n-1)

result = factorial(5)
print(f"The factorial of 5 is {result}")
